# 税务风险管理丨税企分析与税务筹划，以教育培训行业为例

江苏法德东恒律师事务所专职律师，专注领域：公司业务、财税筹划、民商事争议解决。微信：FANTAST_5。“税务风险”一词近年来在企业破产热点话题中经常处于“霸榜”状态，为了更好地吃透税务风险管理问题，无讼研究院特别联合江苏法德东恒律师事务所吴新悦律师打造了“税务风险管理”系列专栏文章，本专栏从破产清算这一特殊程序入手，阐述税务在企业生命运转中不可忽视的作用，同时对税企之间常见的涉税争议热点问题进行分析，最终对企业税务风险管理做出建议，以期对不同纳税人带来相应的启示。本文系专栏文章的第五篇：税务风险管理丨税企分析与税务筹划——教育培训行业。教育培训行业是“大教育”的组成部分，根据培训内容的不同，可将市场上现有的培训分为语言培训、企业培训、IT培训、职业资格培训、考试培训等。教育培训行业的发展深受国家政策的有影响，且随着社会、经济的发展以及政策的革新，教育培训行业的发展呈现一波三折。当然，在这一波三折的发展过程，受诸多因素影响，不考虑外部干扰，更重要的是内部合规，其中税务合规始终处于不可忽视的地位。一、教育培训行业涉及的税种及税务问题对我国现存教育培训机构进行分类，可以将其分为民办非企业的“教育培训机构”与民办企业的“教育培训机构”。民办非企业“教育培训机构”指的是在民政部门办理等级的社会团体或社会组织，领取持有的是《社会团体证书》；民办企业“教育培训机构”指的是在工商部门注册登记的企业、个体户或者其他经济组织，领取持有的是《营业执照》。不同性质的教育培训行业所缴纳的税种有所不同，本文仅对民办企业教育培训机构一般性税种进行分析。（一）增值税部分教育培训机构的税收与一般公司具有相通性，在增值税税目方面，目前的税率是属于生活服务中的教育医疗服务，税率为6%。市场上存在一些教育培训机构在提供教育培训的同时，还提供咨询服务，该服务属于现代服务中的鉴证咨询服务，税率也是6%。若是连续12个月提供教育培训销售额不足500万元的，则属于小纳税人，适用3%的增值税征收率。在增值税缴纳工作中，机构内部财务运作，为减少成本、谋取利益等原因，常常会违规操作，置税务风险而不顾，从而导致机构面临处罚甚至是刑罚风险。常见税务违规操作如下：1.隐匿增值税应税收入。教育培训机构通常以隐匿未开票收入为主要操作方式达到隐匿增值税应税收入的目的，除此以外，培训机构内部员工在处理项目收付款时，由于内部运作不规范，常常出现通过个人账户收款结算的情况，以及以退课、退学等方式随意调节收入的情况，亦会造成增值税应税收入不实的后果。2.确认增值税应税收入不及时。若考虑主观意图，此种税务风险往往是非故意实施行为所造成的隐藏风险，多由于培训机构内部工作处理不及时，如，在收取学员的培训费、课时费、咨询费后，未及时处理、更新账目信息，将其一直挂在预收账款科目，直至培训结束才申报增值税收入；或者是学员已经支付全款培训费用，但未完成培训课程，其长期未上课被培训机构取消课时，但不符合退费条件未办理退费的，其账目未结转收入，以上行为均会产生增值税应税收入不及时之风险。3.增值税收入申报混用税率。培训机构在实际运营中，其所申报税税率会因实际行为性质不同而有所不同，如单独销售教育类相关产品，按照13%申报税率，提供配套教学服务的相关讲义教材按照6%申报税率。而培训机构在实际申报税率时，常常将税率进行混同使用进行申报，由此造成了税务不实问题。（二）企业所得税部分同其他企业相同，教育培训机构的企业所得税缴纳与其他企业的企业所得税缴纳方式相通，企业所得税的交税方式一般有两种，即定期定额缴纳和实际缴纳。定期定额缴纳是指每月按一定比例缴纳企业所得税，根据所得税法，每个月收入低于10万元的可以按月缴纳，收入高于10万元的需要每季度缴纳，比例为每月10%或每季度30%。实际缴纳是指企业根据自身的收入和支出，实际缴纳应缴所得税。企业所得税缴纳需要遵守相关法律法规，不同的企业所得税缴纳方式也有所不同，在企业所得税缴纳过程中，常遇到的税务问题主要有：1.隐匿所得税应税收入。同隐匿增值税相同，教育培训机构主要通过隐匿未开票收入；通过个人账户结算；以退课、退学等方式随意调节收入等方式隐匿所得税应税收入。2.所得税收入确认不及时。所得税确认不及时，主要是教育培训机构在经营过程中对于预收学费或者其他应计为所得税税种科目下的费用，长期挂在应收账款上未进行处理，或者即使学员已经开始上课，对于其收费也不按课时结转收入，由此造成所得税收入确认不及时的问题。除确认不及时之外，还存在对特殊所得税收入不进行确认的情况，如对捐赠收入、无法偿付的应付款项不予确认，由此造成税务未确认问题。3.应税收入确认不完整。对应税收入确认的不完整，在教育培训机构的运营中主要包括存在不合规的佣金、手续费冲减收入，通过没有中介资质的机构或者个人作为招生代表进行招生，然后支付相应的佣金、手续费，或者为了增加学员数量，给予招生回扣,将此部分支出直接冲减收入，导致应纳税所得额减少。（三）个人所得税依据《中华人民共和国个人所得税法》，对于个人所得税的计算，主要包括的计算项目包括居民个人的综合所得；非居民个人的工资、薪金所得；经营所得；财产租赁所得；财产转让所得；利息、股息、红利所得和偶然所得。在教育培训机构的实际运营中，涉及的个人所得税部分主要是薪资所得、偶然所得、利息、股息、红利所得，容易产生的个人所得税问题主要是通过第三方劳务公司补差工资，通过发票报销发放工资的方式减少工资的计发；除此之外，在偶然所得领域，容易出现税务问题的主要是机构运营过程中对于一些诸如礼品赠送、奖金发放之类的偶然所得未按照所得代扣代缴个税。（四）其他税种及税务问题除以上主要税种以及税务问题以外，还存在房产税、城镇土地使用税的缴纳备受关注。主要涉及的、并且需要高度关注的税务问题主要包括：教育培训机构无偿使用学校免税场所进行营利性培训，未缴纳房产税；无租使用集团房屋可能未代缴房产税；纳税单位无偿使用免税单位的土地，可能未按规定缴纳土地使用税；因未取得土地使用证而未申报缴纳土地使用税，土地实际面积与土地使用证存在差异情况下，可能未按照土地实际面积缴纳土地使用税。二、教育培训行业的涉税风险及防范措施本文所阐述的涉税风险仅为侠义的税务风险，即纳税人因未能正确、有效遵守税法而导致企业未来利益的可能损失，包括因没有遵守税法可能遭受的法律制裁、财务损失或声誉损害，具体体现为补税、加收滞纳金、税收行政处罚、税收刑事处罚、信用和商誉下降等。根据前文提及的特定税种中衍生的税务问题，可以将税务风险归为以下几类：（一）行政风险因隐瞒收入、虚列成本面临补税、收取滞纳金、罚款的风险。此种税务风险主要的表现形式为作为纳税人的教育培训机构通过隐瞒收入、虚列成本等方式不进行纳税申报，不缴或者少缴应纳税款。由此产生的法律责任主要是由税务机关追缴其不缴或者少缴的税款、滞纳金，并处不缴或者少缴的税款50%至5倍的罚款。从主观善意角度考虑，如果教育培训机构无法判断业务是否需要申报纳税，建议及时咨询税务机关，与此同时，需要注意纳税申报的期限以及纳税申报的准确，从而主动规避风险。（二）刑事风险偷税、逃税所面临的刑事责任风险。偷税与逃税属于两种不同的行为，其表现形式自然不同。偷税是指纳税人伪造、变造、隐匿、擅自销毁帐簿、记帐凭证，或者在帐簿上多列支出或者不列、少列收入,或者经税务机关通知申报而拒不申报或者进行虚假的纳税申报,不缴或者少缴应纳税款的行为。逃税是指纳税人违反税法规定不缴或少缴税款的非法行为。依据《中华人民共和国税收管理法》，纳税人偷税的，由税务机关追缴其不缴或者少缴的税款、滞纳金，并处不缴或者少缴的税款50%至5倍的罚款；构成犯罪（逃避缴纳税款罪）的，依法追究刑事责任。依据《中华人民共和国刑法》，纳税人采取欺骗、隐瞒手段进行虚假纳税申报或者不申报，逃避缴纳税款数额较大并且占应纳税额百分之十以上的，处三年以下有期徒刑或者拘役，并处罚金；数额巨大并且占应纳税额百分之三十以上的，处三年以上七年以下有期徒刑，并处罚金。扣缴义务人采取前款所列手段，不缴或者少缴已扣、已收税款，数额较大的，依照前款的规定处罚。对多次实施前两款行为，未经处理的，按照累计数额计算。有第一款行为，经税务机关依法下达追缴通知后，补缴应纳税款，缴纳滞纳金，已受行政处罚的，不予追究刑事责任；但是，五年内因逃避缴纳税款受过刑事处罚或者被税务机关给予二次以上行政处罚的除外。税务风险产生的原因，对教育培训机构内部而言，可能系教育培训机构内部管理松散，依法纳税意识薄弱，以及税务工作人员专业水平问题；对于教育培训机构外部而言，可能系税务以及其他相关政策变动引起，无论何种原因，对于以上问题及风险的防范与控制至关重要。建议教育培训机构构建税务风险管控体系，企业应当加强对税务相关工作人员税务风险知识的培训，并在税收法律法规的基础上构建科学、系统的税务风险管控体系，以确保企业税务工作的正常开展。同时，提高企业税务相关工作人员的素质，强化企业税务风险意识。三、教育培训行业的税务筹划上文已经对教育培训机构所应缴纳的税种及税务问题进行了阐述，现对以上税种的税务筹划做分析：（一）教育培训机构关于增值税税务筹划1.免税额提升带来的优惠增值税对销售额比较少的纳税人一直都有着免税规定。随着免税额不断提升，免税政策也就具有了越来越大的税务筹划价值。《财政部 税务总局关于明确增值税小规模纳税人免征增值税政策的公告》（财政部税务总局公告2021年第11号）规定：“自2021年4月1日至2022年12月31日，对月销售额15万元以下（含本数）的增值税小规模纳税人，免征增值税。”在作此筹划时，一定要注意政策的适用对象，以免造成误用、错用；除此之外，还要对税款进行适度调整，每个月的销售额不要超过免税标准。2.教育培训机构分立一般纳税人与小规模纳税人所适用税率是不同的，当教育培训机构发展成为一般纳税人后，可以通过机构分立的方式降低适用税率，而且从一般纳税人分立为小规模纳税人，可以解决增值税专用发票的均衡抵扣和防范发票风险的问题。在此种税务筹划方式适用过程中，需保持年应税销售额适度，以及保持所分立的培训机构数量适度。3.延缓增值税纳税义务发生的时间影响增值税纳税义务发生的要素主要包括开票时间、合同约定的付款时间以及实际的付款时间，当以上要素满足任何一个时，纳税义务也就产生了。因此，教育培训机构把握好开票时间、合同约定的付款时间和实际的付款时间，对于控制增值税纳税义务的发生时间，具有重要的作用。（二）教育培训机构关于企业所得税税务筹划1.申请核定征收核定征收是指由税务机关根据纳税人情况，在正常生产经营条件下，对其生产的应税产品查实核定产量和销售额，然后依照税法规定的税率征收税款的一种征税方式。企业所得税的核定征收包括核定所得税率和核定企业所得税税额两种方式。如果教育培训机构的规模较小，会计核算尚不完备，可以申请核定征收。申请企业所得税核定征收，除了能够解决发票管理方面的问题，也能避免因为收款不规范而引发的风险。2.分期缴纳税款教育培训机构受国家政策影响尤其严重，尤其是国家出台双减政策后，对教育培训机构造成了严重影响，直接导致众多萎靡企业破产。在教育培训机构面临高额税务时，如果可以分期缴纳税款、延长缴纳税款的期限，对于降低内部财务压力也是非常有帮助的。在企业所得税中，有很多关于分期纳税的规定，纳税人应当了解这些规定，同时用好这些规定。如果事先没有做好筹划，待税款交完再回过头来向税务机关申请退税，就会非常被动。（三）教育培训机构关于个人所得税税务筹划税务筹划的目的通常有两个：一个是降低应纳税额，另一个是延缓缴纳税款的时间。在降低应纳税额的问题上，通常采用税收洼地策略，即对各地的税收政策保持敏感，并充分应用税收优惠政策，尽可能降低应纳税额。而延缓个人所得税缴纳税款时间的政策主要分为两类：一类是递延纳税，另一类是分期纳税。在这两类政策中有一个共性的问题，就是纳税人必须做好税务备案。税务备案这个看上去例行公事的举动，如果没有做到，就可能使税务筹划前功尽弃。四、相关税收政策1.《国家税务总局关于社会力量办学征收个人所得税问题的批复》(国税函〔1998〕738号)发文机关：国家税务总局发布日期：1998.12.07生效日期：1998.12.07
