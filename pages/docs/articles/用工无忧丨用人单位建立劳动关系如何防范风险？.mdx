# 用工无忧丨用人单位建立劳动关系如何防范风险？

北京市炜衡（西安）律师事务所律师，专注于军工科技、金融投资、房地产等行业的商事争议解决、非诉法律服务和劳动争议法律服务。编者按：劳动用工是市场经济运行中必不可少的一环，劳动用工的风险管理也是企业无法回避的问题。无讼联合北京市炜衡（西安）律师事务所查长宝律师开设「用工无忧」写作专栏，从劳动用工的全流程（招聘+劳动合同订立+劳动合同履行+劳动合同变更解除）出发，聚焦每个用工阶段的核心风险点，融合法律分析的理论视角和实务案例的实务视角，为企业的“用工无忧”提出针对性建议。今天刊发专栏第七篇文章：用人单位建立劳动关系如何防范风险？01风险点风险点一：建立劳动关系不签订书面劳动合同风险点二：签订其他名义合同替代订立劳动合同02法律分析（一）劳动关系如何认定？我国现行法律确立了“以书面劳动合同认定劳动关系为原则，以事实行为认定劳动关系为例外”的规则。《劳动合同法》第七条规定，用人单位自用工之日起即与劳动者建立劳动关系。第十条规定，建立劳动关系，应当订立书面劳动合同。已建立劳动关系，未同时订立书面劳动合同的，应当自用工之日起一个月内订立书面劳动合同。用人单位与劳动者在用工前订立劳动合同的，劳动关系自用工之日起建立。出于对劳动者权益的充分保护，立法者要求用人单位与劳动者建立劳动关系时应当订立书面劳动合同，即明确了订立书面劳动合同是用人单位的法定义务。立法者之所以重视书面劳动合同的订立，是因为书面劳动合同对于劳动关系的核心要素（工作地点、工作岗位、工作时间、工作报酬等）一般会进行约定，如出现劳动纠纷，劳动者更容易举证，其权利能得到更好的保护。订立书面劳动合同是一种强制性要求，是出于保护劳动者权益的考虑，但是不能因未订立书面劳动合同就在法律上否认劳动关系，如果按照此逻辑进行认定，将更不利于劳动者权益的保护，因此立法者又确立了事实劳动关系的认定规则。《关于确立劳动关系有关事项的通知》（劳社部发〔2005〕12号）第一条规定，用人单位招用劳动者未订立书面劳动合同，但同时具备下列情形的，劳动关系成立：(一)用人单位和劳动者符合法律、法规规定的主体资格；(二)用人单位依法制定的各项劳动规章制度适用于劳动者，劳动者受用人单位的劳动管理，从事用人单位安排的有报酬的劳动；(三)劳动者提供的劳动是用人单位业务的组成部分。上述规定明确了未订立书面劳动合同情形下劳动关系认定的实质要件，如满足主体资格、接受管理、支付报酬和业务组成部分，则成立事实劳动关系。第二条明确了认定劳动关系常见的形式要件，即用人单位未与劳动者签订劳动合同，认定双方存在劳动关系时可参照下列凭证：(一)工资支付凭证或记录(职工工资发放花名册)、缴纳各项社会保险费的记录；(二)用人单位向劳动者发放的“工作证”、“服务证”等能够证明身份的证件；(三)劳动者填写的用人单位招工招聘“登记表”、“报名表”等招用记录；(四)考勤记录；(五)其他劳动者的证言等。其中，(一)、(三)、(四)项的有关凭证由用人单位负举证责任。通过上述梳理可知，以书面劳动合同认定劳动关系属于基本原则和基本方式，以事实行为认定劳动关系属于例外情形。由于订立书面劳动合同的本身重要性，立法者将其设定为用人单位的一项重要义务，下文将对未订立书面劳动合同的法律后果进行分析梳理。（二）用人单位不订立书面劳动合同的法律后果？不订立书面劳动合同虽然涉及到用人单位和劳动者双方，但是总体而言是否订立书面劳动合同是由用人单位单方控制的，因而对于未订立书面劳动合同的法律后果亦针对用人单位而专门设置。《劳动合同法》第八十二条确立了“未订立书面劳动合同的双倍工资”规则，即用人单位自用工之日起超过一个月不满一年未与劳动者订立书面劳动合同的，应当向劳动者每月支付二倍的工资。用人单位违反本法规定不与劳动者订立无固定期限劳动合同的，自应当订立无固定期限劳动合同之日起向劳动者每月支付二倍的工资。二倍工资在劳动争议案件中通常属于金额较大的主张，因此应当引起用人单位单位的高度重视，就司法实践而言，与二倍工资相关的以下问题需要厘清：1、二倍工资的时效关于二倍工资时效问题，司法实践中存在四种观点，梳理如下：观点一：二倍工资属于劳动报酬，适用特别时效，即劳动关系终止后一年。观点二：劳动者从入职满一个月次日即知晓侵权事实，即时效应从入职满一个月次日起算一年。观点三：二倍工资是按月计算的，因此时效应当按月份逐个计算，即入职第二月的二倍工资从入职满两个月次日起算一年，入职第十二月的二倍工资从入职满一年次日起算一年。观点四：二倍工资不属于劳动报酬，应适用一般时效，并且未订立书面劳动合同侵权具有持续性，应当自入职满一年次日起算一年。以上四种观点在司法实践中均有出现，从笔者目前检索的案例来看，观点四属于主流观点，在司法实践中的应用范围最广。2、三种情形下的二倍工资上限情形一：入职未订立劳动合同《劳动合同法》第十条规定用人单位应当自用工之日起一个月内订立书面劳动合同，即给予了用人单位一个月的宽限期，另根据《劳动合同法》第十四条“用人单位自用工之日起满一年不与劳动者订立书面劳动合同的，视为用人单位与劳动者已订立无固定期限劳动合同”之规定，入职未订立劳动合同的二倍工资上限为11个月。2022年人力资源社会保障部和最高人民法院发布的《关于劳动人事争议仲裁与诉讼衔接有关问题的意见（一）》第二十条也明确“劳动者主张自用工之日起满一年之后的第二倍工资的不应支持”。情形二：合同期满未续订劳动合同劳动者与用人单位的劳动合同期满后，双方应当续订劳动合同，如未续订则用人单位仍存在支付二倍工资的风险。因合同期满情形下用人单位不存在宽限期，且基于视为订立无固定期限劳动合同的规定，合同期满未续订劳动合同的二倍工资上限为12个月。情形三：未订立无固定期限劳动合同《劳动合同法》第八十二条规定了未订立无固定期限劳动合同支付二倍工资的情形，除“视为订立无固定期限劳动合同”的情形外，其他情形没有明确的上限，二倍工资可计算到实际订立无固定期限劳动合同之日止。但是，由于二倍工资受一年普通仲裁时效的限制，所以实际能支持的二倍工资也无法超过12个月。（三）协议内容与用工行为不一致的法律后果如何认定？在《企业如何选择合适的用工形式？》一文中，笔者对用工协议约定与实际用工行为不一致如何认定进行了分析，即用人单位与劳动者未订立劳动合同，而是签订其他形式的合同（例如劳务协议、顾问协议等），如果双方实际履约行为符合《关于确立劳动关系有关事项的通知》（劳社部发〔2005〕12号）第一条规定的劳动关系实质要件，则双方的权利义务仍应按照劳动关系的法律规则进行处理。而且，由于双方未订立书面劳动合同，劳动关系项下对劳动者约束的条款未能明确，用人单位反而面临更大的管理风险和法律风险。03实务案例实务案例一：聂美兰与北京林氏兄弟文化有限公司确认劳动关系案（2018）京01民终5911号（指导案例179号）——用人单位与劳动者签订的书面协议符合劳动合同条款，劳动者以用人单位未订立书面劳动合同为由要求支付第二倍工资的不予支持。主要事实：2016年4月8日，聂美兰与北京林氏兄弟文化有限公司（以下简称林氏兄弟公司）签订了《合作设立茶叶经营项目的协议》，内容为：“第一条：双方约定，甲方出资进行茶叶项目投资，聘任乙方为茶叶经营项目经理，乙方负责公司的管理与经营。第二条：待项目启动后，双方相机共同设立公司，乙方可享有管理股份。第三条：利益分配：在公司设立之前，乙方按基本工资加业绩方式取酬。公司设立之后，按双方的持股比例进行分配。乙方负责管理和经营，取酬方式：基本工资＋业绩、奖励＋股份分红。第四条：双方在运营过程中，未尽事宜由双方友好协商解决。第五条：本合同正本一式两份，公司股东各执一份。”协议签订后，聂美兰到该项目上工作，工作内容为负责《中国书画》艺术茶社的经营管理，主要负责接待、茶叶销售等工作。林氏兄弟公司的法定代表人林德汤按照每月基本工资10000元的标准，每月15日通过银行转账向聂美兰发放上一自然月工资。聂美兰请假需经林德汤批准，且实际出勤天数影响工资的实发数额。2017年5月6日林氏兄弟公司通知聂美兰终止合作协议。聂美兰实际工作至2017年5月8日。聂美兰申请劳动仲裁，认为双方系劳动关系并要求林氏兄弟公司支付未签订书面劳动合同二倍工资差额，林氏兄弟公司主张双方系合作关系。北京市海淀区劳动人事争议仲裁委员会作出京海劳人仲字（2017）第9691号裁决：驳回聂美兰的全部仲裁请求。聂美兰不服仲裁裁决，于法定期限内向北京市海淀区人民法院提起诉讼。裁判观点：申请人林氏兄弟公司与被申请人聂美兰签订的《合作设立茶叶经营项目的协议》系自愿签订的，不违反强制性法律、法规规定，属有效合同。对于合同性质的认定，应当根据合同内容所涉及的法律关系，即合同双方所设立的权利义务来进行认定。双方签订的协议第一条明确约定聘任聂美兰为茶叶经营项目经理，“聘任”一词一般表明当事人有雇佣劳动者为其提供劳动之意；协议第三条约定了聂美兰的取酬方式，无论在双方设定的目标公司成立之前还是之后，聂美兰均可获得“基本工资”“业绩”等报酬，与合作经营中的收益分配明显不符。合作经营合同的典型特征是共同出资，共担风险，本案合同中既未约定聂美兰出资比例，也未约定共担风险，与合作经营合同不符。从本案相关证据上看，聂美兰接受林氏兄弟公司的管理，按月汇报员工的考勤、款项分配、开支、销售、工作计划、备用金的申请等情况，且所发工资与出勤天数密切相关。双方在履行合同过程中形成的关系，符合劳动合同中人格从属性和经济从属性的双重特征。故原判认定申请人与被申请人之间存在劳动关系并无不当。双方签订的合作协议还可视为书面劳动合同，虽缺少一些必备条款，但并不影响已约定的条款及效力，仍可起到固定双方劳动关系、权利义务的作用，二审法院据此依法改判（林氏兄弟公司无需支付二倍工资）是正确的。实务案例二：卢瑞其、漳州招商房地产有限公司劳动争议案(2021)闽02民终262号——劳动合同到期后未续签书面劳动合同，用人单位有支付二倍工资的风险主要事实：2017年9月28日，卢瑞其入职漳州招商房地产有限公司。2017年9月30日，签订书面劳动合同，合同约定的工作岗位为客服负责人，合同期满时间为2019年9月30日。2020年4月17日漳州招商公司单方解除劳动关系。劳动合同期满后劳动者工资情况：2019年10月工资18514元，2019年11月至2020年2月每月工资18700元，2020年3月工资17867元，2020年4月份工资按整月发放19532元。裁判观点：双方劳动合同到期后，卢瑞其仍然在漳州招商公司上班，漳州招商公司与卢瑞其之间构成用工关系，但漳州招商公司作为用人单位，未能提供充分的证据证明，在卢瑞其拒绝其单方违法变更劳动合同内容的决定后，有与卢瑞其协商妥善解决争议，或依法作出处理，而是放任劳动关系的僵局持续数月，以致未依法与卢瑞其签订书面劳动合同，一审法院依法判决漳州招商公司支付未签订书面劳动合同的二倍工资差额，并无不当。【备注】实践中有案例认为劳动合同期满后，劳动者仍在原用人单位工作，原用人单位未表示异议的，视为双方同意以原条件继续履行劳动合同，该情形不应适用《劳动合同法》第八十二条的二倍工资。实务案例三：凯荣置业（沈阳）有限公司与白玉红劳动争议纠纷（2015）沈中民五终字第1107号——两次固定期限合同后，仍签订固定期限合同，不一定需支付二倍工资。主要事实：2008年9月1日，白玉红到凯荣置业（沈阳）有限公司（以下简称凯荣置业）工作。2013年9月26日，双方解除劳动关系。上述期间，双方连续订立了四次固定期限劳动合同。第一次2008年9月1日订立，合同期限2008年9月1日至2011年8月31日；第二次2011年8月31日订立，合同期限2011年9月1日至2012年8月31日；第三次2012年9月1日订立，合同期限2012年9月1日至2013年8月31日；第四次2013年8月31日订立，合同期限2013年9月1日至2014年8月31日。2013年11月11日，白玉红提起劳动仲裁，要求凯荣置业给付未签订无固定期限劳动合同的二倍工资等仲裁请求，未获支持。白玉红不服，诉至一审法院。裁判观点：关于未签订无固定期限劳动合同双倍工资的问题。《中华人民共和国劳动合同法》第十四条规定：“无固定期限劳动合同，是指用人单位与劳动者约定无确定终止时间的劳动合同。用人单位与劳动者协商一致，可以订立无固定期限劳动合同。有下列情形之一，劳动者提出或者同意续订、订立劳动合同的，除劳动者提出订立固定期限劳动合同外，应当订立无固定期限劳动合同：（三）连续订立二次固定期限劳动合同，且劳动者没有本法第三十九条和第四十条第一项、第二项规定的情形，续订劳动合同的。”本案中，双方实际已经连续签订了四次固定期限劳动合同，在双方已经连续签订两次固定期限劳动合同后，劳动者应提出与用人单位签订无固定期限劳动合同，此时用人单位与劳动者签订无固定期限劳动合同就成了用人单位的义务，但劳动者仍选择与用人单位签订固定期限劳动合同，则意味着劳动者已经放弃了与用人单位签订无固定期限劳动合同的权利，同时这也是劳动者自由行使劳动权利的体现。现劳动者并未向法院提交充分证据证明其曾向用人单位提出过签订无固定期限劳动合同，应承担举证不能的后果，应视为劳动者对连续两次签订固定期限后仍签订后续固定期限劳动合同的同意。故在上述情形下劳动者提出的未签订无固定期限劳动合同双倍工资的请求不应支持。故一审法院对该项判决有误，本院予以更正。实务案例四：刘丹萍与南京仁创物资有限公司劳动争议纠纷案(2015)江宁民初字第4477号（公报案例）——人事主管负责管理订立劳动合同的， 用人单位无需支付其未订立书面劳动合同二倍工资。主要事实：刘丹萍于2015年3月10日进入被告仁创公司工作，担任人事主管，主要负责人员招聘、培训及薪酬管理工作。2015年7月23日刘丹萍离开仁创公司，并于同日以仁创公司未与其签订劳动合同，未依法给予其哺乳期内每天1小时的哺乳时间，且未足额支付其月度工资严重侵害其合法权益为由，向仁创公司寄送《关于与南京仁创物资有限公司解除劳动关系的通知函》，通知仁创公司自2015年7月24日起解除双方的劳动关系，并要求仁创公司给予补偿。原告刘丹萍于2015年7月24日向南京市江宁区劳动人事争议仲裁委员会申请仲裁，要求仁创公司足额发放2015年3月至7月预留工资4165元、自入职之日起未签订劳动合同的双倍工资11666元及哺乳时间工资3385元以及经济补偿金2917元，合计22133元。裁判观点：关于刘丹萍主张的双倍工资11666元，本案中，仁创公司确实未与刘丹萍签订劳动合同，但是，一般而言，劳动合同签订事项属于人力资源负责的事项，刘丹萍作为仁创公司的人事主管，其工作职责范围应该包括代表单位依照法律法规处理与劳动者之间劳动合同履行方面的相关事宜，避免单位因违反法律法规被追究法律责任，也应当知晓订立书面劳动合同的相关规定及不订立书面劳动合同的法律后果，因此，刘丹萍有义务主动向仁创公司要求订立书面劳动合同。但刘丹萍并未提交证据证明其曾主动要求仁创公司与其签订劳动合同。故综合刘丹萍的岗位职务因素等考量后，本院对刘丹萍主张的未签订劳动合同的双倍工资的请求不予支持。04无忧指南（一）制定劳动合同范本，用工之日订立书面劳动合同1、为了提升管理的效率和降低用工风险，用人单位可根据本单位的实际情况制定劳动范本。2、为了避免遗漏签订书面劳动合同，用人单位可将书面劳动合同的签订作为入职流程表中的一项，并且尽可能在入职当天完成。3、员工签订书面劳动合同应当有用人单位的工作人员在场，确保劳动者是亲自签订。（二）关注劳动合同到期时间，做好书面劳动合同续签工作1、用人单位可通过人工或者技术手段设置劳动合同的到期提醒。2、在劳动合同到期前评估是否续签，如果决定解除劳动合同不要延续到期满之后。3、如果确定继续劳动关系，一定要签订书面劳动合同。（三）定期梳理劳动合同状态，完成无固定期限劳动合同的协商和签订1、用人单位应在每份劳动合同到期前梳理劳动合同的性质，并确定是否满足无固定期限劳动合同的法定标准。2、在签订无固定期限劳动合同前，可征求员工关于劳动合同类型的意见，如员工同意签订固定期限劳动合同，应留存书面协商记录；如员工要求签订无固定期限劳动合同，应当订立书面无固定期限劳动合同。（四）规范劳动关系协议签订，避免法律关系含混不清1、如用人单位想与劳动者建立其他用工关系而非劳动关系，建议咨询律师签订相关性质的协议并关注履约过程中的注意事项。2、如用人单位拟与劳动者建立劳动关系，应当签订完备劳动合同，谨慎通过其他类型合同替代劳动合同，以尽可能通过合同约束劳动者，维护用人单位权益。05相关规范1、《劳动合同法》第七条 用人单位自用工之日起即与劳动者建立劳动关系。用人单位应当建立职工名册备查。第十条 建立劳动关系，应当订立书面劳动合同。已建立劳动关系，未同时订立书面劳动合同的，应当自用工之日起一个月内订立书面劳动合同。用人单位与劳动者在用工前订立劳动合同的，劳动关系自用工之日起建立。第十四条 用人单位自用工之日起满一年不与劳动者订立书面劳动合同的，视为用人单位与劳动者已订立无固定期限劳动合同。第八十二条 用人单位自用工之日起超过一个月不满一年未与劳动者订立书面劳动合同的，应当向劳动者每月支付二倍的工资。用人单位违反本法规定不与劳动者订立无固定期限劳动合同的，自应当订立无固定期限劳动合同之日起向劳动者每月支付二倍的工资。2、《关于确立劳动关系有关事项的通知》一、用人单位招用劳动者未订立书面劳动合同，但同时具备下列情形的，劳动关系成立。(一)用人单位和劳动者符合法律、法规规定的主体资格；(二)用人单位依法制定的各项劳动规章制度适用于劳动者，劳动者受用人单位的劳动管理，从事用人单位安排的有报酬的劳动；(三)劳动者提供的劳动是用人单位业务的组成部分。二、用人单位未与劳动者签订劳动合同，认定双方存在劳动关系时可参照下列凭证：(一)工资支付凭证或记录(职工工资发放花名册)、缴纳各项社会保险费的记录；(二)用人单位向劳动者发放的“工作证”、“服务证”等能够证明身份的证件；(三)劳动者填写的用人单位招工招聘“登记表”、“报名表”等招用记录；(四)考勤记录；(五)其他劳动者的证言等。其中，(一)、(三)、(四)项的有关凭证由用人单位负举证责任。3、人力资源社会保障部 最高人民法院关于劳动人事争议仲裁与诉讼衔接有关问题的意见（一）二十、用人单位自用工之日起满一年未与劳动者订立书面劳动合同，视为自用工之日起满一年的当日已经与劳动者订立无固定期限劳动合同。存在前款情形，劳动者以用人单位未订立书面劳动合同为由要求用人单位支付自用工之日起满一年之后的第二倍工资的，劳动人事争议仲裁委员会、人民法院不予支持。无讼小编：如果您觉得这篇文章还不错，欢迎转发分享、点赞收藏，您也可以在下方评论区留下自己的观点，和大家一起讨论。主编：靖力责编：梁萌审核：刘逸凡
