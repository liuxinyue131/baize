# 国企改革丨国有企业重大资产交易实务要点

企业房产、机动车、特种车辆、在建工程、土地使用权和一定金额以上的债权、知识产权、生产设备及其他资产等对外转让，应当按照企业内部管理制度履行相应决策程序后，在北交所公开进行[5]。2018年1月，四川省国资委出台了《四川省企业国有资产交易监督管理办法》，明确了企业转让资产账面净值或评估价值100万元以上的，应当在产权交易机构公开进行[6]。2020年6月，湖南省国资委、财政厅下发《关于省属企业国有资产交易监督管理有关事项的通知》，规定了省属国企的全部国有资产交易应在依法设立和确定的产权交易机构中公开进行[7]。2021年11月，贵州省国资委于出台了《贵州省企业国有资产交易监督管理办法》，对于须在产权交易机构公开挂牌进行转让的资产金额作出了“一次转让资产评估值在100万元以上”的限定[8]。02涉及的主要法律风险从国有企业方视角来看，在国有企业重大资产交易中涉及的法律风险主要包括以下四项：风险一：对合同效力发生误判，导致交易结果与预期严重不符一种情况是，国有企业方主观上想达成交易，但因未履行完相关程序致合同未生效，导致国企承担缔约过失责任或付出更多经济成本。另一种情况，国有企业方可能主观上认为该交易需要等待审批暂缓进度，但因履行了签约程序，致使交易必须进行或在无法履约的情况下承担违约责任。其关键点就在于国有资产交易中如果未经政府审批有可能导致合同不生效，但并非必然导致合同不生效。经检索相关判例发现，是否经政府批准对合同效力的影响主要分为以下几类：1.转让部分国有资产致使国家对该企业不再具有控股地位的，应当报请本级人民政府批准，依据主流裁判观点，未报批准的该资产转让行为不生效。在最高院审理的沈某、江苏省国信集团有限公司股权转让纠纷再审审查与审判监督一案中，法院认为，《企业国有产权转让管理暂行办法》中关于转让企业国有产权致使国家不再拥有控股地位的应报本级人民政府批准、重要子企业的重大国有产权转让事项应报同级国有资产监督管理机构会签财政部门后批准的规定，体现了对于国有资产转让程序的统一管理。原审判决认定在涉及国有资产转让的股权转让中应当依照上述规范性文件的要求办理正确……《最高人民法院关于适用<中华人民共和国合同法>若干问题的解释（一）》第九条规定，法律、行政法规规定合同应当办理批准手续，或者办理批准、登记等手续才生效，在一审法庭辩论终结前当事人仍未办理批准手续的，或者仍未办理批准、登记等手续的，人民法院应当认定该合同未生效……故原审判决认定《股权转让合同》未生效，符合法律规定[9]。2.任何单位和个人购买商业银行股份总额百分之五以上的，应当事先经国务院银行业监督管理机构批准，依据主流裁判观点，未经批准的该购买股份行为不生效。在最高院审理的中信红河矿业有限公司、鞍山市财政局股权转让纠纷一案中，法院认为，《中华人民共和国商业银行法》第二十八条规定，任何单位和个人购买商业银行股份总额百分之五以上的，应当事先经过国务院银行业监督管理机构批准。涉案《股份转让合同书》的转让标的为鞍山财政局持有的鞍山银行7.9989%即20000万股股权，系金融企业国有资产，转让股份总额已经超过鞍山银行股份总额的5%。依据上述规定,该合同应经有批准权的政府及金融行业监督管理部门批准方产生法律效力。由此，本案的《股份转让合同书》虽已经成立，但因未经有权机关批准，应认定其效力为未生效[10]。3.依据主流裁判观点，国企转让债权未经政府审批不影响合同效力认定。在济南中院审理的汉盛控股集团有限公司与济南荣协国际贸易有限公司等债权转让合同纠纷一案中，法院认为，《企业国有资产监督管理暂行条例》等规范性文件均未对国有控股企业对外转让债权需由所出资企业报国有资产监督管理机构批准作出相关规定，故汉盛公司对外转让债权不应适用《合同法》第四十四条第二款的规定。汉盛公司与荣协公司、京胜公司签订的《债权转让协议》系各方当事人真实意思表示，合同内容不违反法律和行政法规的强制性规定，不违背公序良俗，合法有效。一审法院认定《债权转让协议》因未履行批准手续未生效不当，本院予以纠正[11]。风险二：未对资产进行评估或评估存在重大失误，未做尽职调查或尽职调查不充分，均有可能导致资产价值与交易对价差异过大，产生国有资产损失的风险在国有企业重大资产交易中，评估原则上是交易必经程序，而尽职调查则是国有企业对外合作投资及收购资产的必备合规程序。根据《企业国有资产法》之规定，国有企业合并、分立、改制，转让重大财产，以非货币财产对外投资，清算或者有法律、行政法规以及企业章程规定应当进行资产评估的其他情形的，应当按照规定对有关资产进行评估。同时，依据《国务院办公厅关于建立国有企业违规经营投资责任追究制度的意见》，国有企业经营管理有关人员在投资并购未按规定开展尽职调查，或尽职调查未进行风险分析等，存在重大疏漏造成国有资产损失以及其他严重不良后果的，应当追究责任。风险三：在国有企业重大资产交易中，未履行相关程序或在交易中有重大过失致使国有资产受损的，国有企业和相关人员有涉及行政责任的风险根据《财政违法行为处罚处分条例》之规定，国家机关及其工作人员违反国有资产管理的规定，擅自占有、使用、处置国有资产的，责令改正，调整有关会计账目，限期退还违法所得和被侵占的国有资产。对单位给予警告或者通报批评。对直接负责的主管人员和其他直接责任人员给予记大过处分；情节较重的，给予降级或者撤职处分；情节严重的，给予开除处分。该条例还明确了“企业、事业单位、社会团体中由国家行政机关以委任、派遣等形式任命的人员以及其他人员有本条例规定的财政违法行为，需要给予处分的，参照本条例有关规定执行。”同时，依据《国务院办公厅关于建立国有企业违规经营投资责任追究制度的意见》，国有企业经营管理有关人员存在以下一些情况，如在转让产权、上市公司股权和资产方面未按规定履行决策和审批程序或超越授权范围转让，财务审计和资产评估违反相关规定；如在投资并购方面未按规定开展尽职调查，或尽职调查未进行风险分析等，存在重大疏漏，财务审计、资产评估或估值违反相关规定，或投资并购过程中授意、指使中介机构或有关单位出具虚假报告，未按规定履行决策和审批程序，决策未充分考虑重大风险因素等，造成国有资产损失以及其他严重不良后果的，应当追究责任。风险四：在国有企业重大资产交易中，因故意或重大过失造成国家利益遭受损失的，相关人员可能构成刑事犯罪在国有企业重大资产交易中，通常最容易让人想到的后果就是造成国有资产流失，其实在我国《刑法》中并无“国有资产流失罪”这个罪名，但造成国有资产损失确实是认定国有企业人员触犯相关犯罪的重要构成要件。在国有企业重大资产交易中，除贪污贿赂类犯罪外主要涉及以下几类刑事罪名：1.签订、履行合同失职被骗罪。《刑法》第一百六十七条规定，国有公司、企业、事业单位直接负责的主管人员，在签订、履行合同过程中，因严重不负责任被诈骗，致使国家利益遭受重大损失的，处三年以下有期徒刑或者拘役；致使国家利益遭受特别重大损失的，处三年以上七年以下有期徒刑。2.国有公司、企业、事业单位人员失职罪，国有公司、企业、事业单位人员滥用职权罪。《刑法》第一百六十八条规定，国有公司、企业的工作人员，由于严重不负责任或者滥用职权，造成国有公司、企业破产或者严重损失，致使国家利益遭受重大损失的，处三年以下有期徒刑或者拘役；致使国家利益遭受特别重大损失的，处三年以上七年以下有期徒刑。国有事业单位的工作人员有前款行为，致使国家利益遭受重大损失的，依照前款的规定处罚。国有公司、企业、事业单位的工作人员，徇私舞弊，犯前两款罪的，依照第一款的规定从重处罚。3.徇私舞弊低价折股、出售国有资产罪。《刑法》第一百六十九条规定，国有公司、企业或者其上级主管部门直接负责的主管人员，徇私舞弊，将国有资产低价折股或者低价出售，致使国家利益遭受重大损失的，处三年以下有期徒刑或者拘役；致使国家利益遭受特别重大损失的，处三年以上七年以下有期徒刑。03主要问题及操作建议在实务操作中，因交易主体不同、资产类型不同，基于平衡风险与效率的需要，企业常常对国有资产交易产生一些疑惑，比较常见的疑问有：定价程序必须以评估为基础吗？是否国有资产交易都需要进行尽职调查？是否所有国有资产交易都必须进场交易？交易主体都属国有单位的情况下是否也要进行评估、进场？国有资产交易原则需要经历决策程序、定价程序和交易程序三个环节，根据《企业国有资产法》《企业国有资产交易监督管理办法》《企业国有资产监督管理暂行条例》《国有资产评估管理办法》等法律法规之要求，以实务中的具体问题为出发点，笔者建议国有企业和律师把控好以下一些关键点。关键环节一：区分资产交易类型第一步先确认资产交易的类型，主要从这几方面进行区分：是收购资产还是处置资产，交易对方是国有主体还是民营主体或自然人，涉及交易金额预估多少等等。确定交易类型有助于选择比较合适的交易方式及路径。关键环节二：决策程序注意事项根据《关于进一步推进国有企业贯彻落实“三重一大”决策制度的意见》之规定，国有企业重大资产交易属于“重大决策事项”，应依照相关党内法规、公司章程、公司内部决策制度之要求提交该国有企业党组织、经理层、董事会/执行董事、股东会/股东等进行审议决策。同时，在国有资产交易中，决策程序不仅是合规要求，很多情况下也涉及法律效力性要求。根据《企业国有资产法》第五十三条之规定，履行出资人职责的机构决定转让全部国有资产的，或者转让部分国有资产致使国家对该企业不再具有控股地位的，应当报请本级人民政府批准。根据《民法典》第五百零二条“依法成立的合同，自成立时生效……依照法律、行政法规的规定，合同应当办理批准等手续的，依照其规定”，前述交易未经批准的合同不生效。需要注意的是，交易主体切勿轻易将“未经政府批准合同不生效”当做交易策略。一方面，如本文第二部分所述，并非所有国有企业重大资产交易合同未经批准都不生效；另外，就算交易合同不生效，过失方也可能因此承担缔约过失责任。关键环节三：定价程序注意事项资产评估是国有资产交易的法定程序，在交易中原则以资产评估结果为定价依据。从立法目的上来看，资产评估是国有资产交易定价、判断国有资产增值与否、财务记账的依据，也是防止国有资产损失的重要措施，因此包括《企业国有资产法》《企业国有资产交易监督管理办法》《国有资产评估管理办法》都明确了国有资产交易的评估要求。除企业国有产权在政府机构、事业单位、国有独资企业、国有独资公司之间的无偿划转可以不经评估，“依据中介机构出具的被划转企业上一年度（或最近一次）的审计报告或经国资监管机构批准的清产核资结果，直接进行账务调整，并按规定办理产权登记等手续”[12]等少数情形外，资产评估几乎是国有资产交易合规绕不开的必经环节。关键环节四：交易程序注意事项交易程序需要从转让和购买两个方面来说。如果是转让国有资产原则上应该进场交易，根据《中华人民共和国企业国有资产法》第五十四条之规定，除按照国家规定可以直接协议转让的以外，国有资产转让应当在依法设立的产权交易场所公开进行。也就是说国有资产处置原则需要进场交易，例外情况下才可以协议转让，否则资产转让协议有可能被认定为无效[13]。如果是国有企业对外购买重大资产，如对方是非国有主体则没有必须进场交易的要求，原则上以资产评估为依据结合尽职调查结果进行定价，双方履行完各自决策程序后即可完成交易。国有企业重大资产交易属于对程序与实体要求均非常之高的交易类型，需要企业与律师、注册会计师等第三方机构共同审慎把关好决策程序、定价程序和交易程序等环节，防范好交易事项涉及的合规风险与法律风险。注释：[1]
