# 证券刑事丨一人负天下之人——背信损害上市公司利益罪的解析及辩护策略

北京天驰君泰律师事务所上海分所合伙人按语：2019年以来，随着《证券法》的重新修订，以及《最高人民法院、最高人民检察院、公安部、中国证监会关于办理证券期货违法犯罪案件工作若干问题的意见》等司法解释的重磅出台，过去仅止步于行政处罚的证券类违法案件中，有相当数量被移送至公安机关刑事立案处理。相关数据显示，2021年人民法院依法严惩金融证券犯罪，审结操纵市场、内幕交易、非法集资、洗钱等犯罪案件1.3万件。这足见我国对证券类犯罪的打击力度之大，与十几年前相比，已不可同日而语。证券类犯罪是刑事领域最为复杂的分支之一。在目前证券市场“强监管”“零容忍”的背景下，辩护律师的充分参与，有助于证券类案件得到更客观、更公正、更全面、更专业的处理，从而依法维护委托人的合法权益。本文系“证券刑事”第五篇：证券刑事丨一人负天下之人——背信损害上市公司利益罪的解析及辩护策略。人而无信，不知其可也。大车无輗，小车无軏，其何以行之哉[1]。——《论语·为政》传统社会的儒家价值观推崇的是“仁义礼智信”。义薄云天、忠肝义胆的关羽被奉为神明，至今民间香火不断；而“宁可我负天下人，不可天下人负我”的曹操，千百年来，在戏剧舞台上都是白脸奸臣，直至清代才有毛宗岗等人开始为其“平反”。受传统文化影响，“背信”二字在中文的语境中，尤其会带给人不快之感。当这个词映入眼帘，你或许会想到某个借了东西不还的失联同学，某个说话不算数的断交朋友，某个无故撕毁合同的交易伙伴。上述这些不讲信用的林林总总，绝大多数只能受到道德和舆论的谴责，部分则是民事上的违约行为，可能涉及诉讼仲裁。毕竟，如果轻易动用刑事手段，对个体偶发的背信行为进行惩戒，颇有“杀鸡焉用牛刀”之感，也有违刑法的谦抑性。除非，此人像古人眼中的曹操一样，凭借一己之力负了天下之人。一人如何辜负天下之人？一己之罪过，如何直接损害万千人的利益？我国现行《公司法》规定，董事、监事、高级管理人员对公司负有忠实义务和勤勉义务。既然上市公司是公众的公司，上市公司的董事、监事、高级管理人员系受众人之托开展工作，如违背忠实勤勉之义务，如果背信损害上市公司利益，情节严重的将接受刑法裁决，构成背信损害上市公司利益罪。一、背信损害上市公司利益罪的设置背景我国的全民所有制企业，即国有企业天然具有公众性质，因此1997年修订的《刑法》即规定了“为亲友非法牟利罪”和“非法经营同类罪”，以打击通过向特殊关系人输送利益的方式，侵害国有企业利益的不法行为。随着证券市场的发展，上市公司数量与日俱增，对同样具有公众性质的上市公司，特别是对于上市公司中的民营企业，在刑法上进行同等的保护，避免众多损害上市公司利益的行为因不符合其他罪名的构成要件而无法被追究刑事责任，亦成为时代呼声和法治社会的客观需求。因此，为了维护社会主义市场经济秩序，以保护国有企业的力度，维护上市公司合法权益，2006年通过的《刑法修正案（六）》在“破坏社会主义市场经济秩序罪”章节增设了第169条之一的“背信损害上市公司利益罪”，对公司高层的背信行为进行了刑法规制，意图惩治违背对公司的忠实义务，利用职务便利，通过关联交易进行利益输送等方式，损害上市公司利益的行为。上市公司均为股份有限公司，且大多数实行所有权与经营权分离的经营模式，但此种模式无法完全杜绝控股股东、实际控制人利用其控股地位或实控权利，通过关联交易等方式损害上市公司利益，并最终侵害中小股东的权益。因此，刑法明确规定，上市公司的控股股东或者实际控制人，指使上市公司董事、监事、高级管理人员实施上述背信行为的，也可以该罪追究刑事责任。二、背信损害上市公司利益罪的内涵理解背信损害上市公司利益罪的内涵，首先，我们需要对“背信”二字进行法律层面的解读。上海政法学院刑事司法学院长彭华文教授认为，根据主观心理差异，可将主观背信分为三种类型：一是草率型背信（过失型背信），即因未能充分了解合约风险，或者在履行合约过程中疏忽，致使履约不能或者不正当履约的情形。二是故意型背信，即主观上无视双方约定，不正当履行义务，损害他人利益的情形。三是欺骗型背信，即基于非法占有目的，采取虚构事实、隐瞒真相的方式签约，损害他人利益的情形。并非任何背信行为都可被认定为背信犯罪。对草率型背信进行归罪，可能会使当事人忌惮刑法苛责而选择退出或者不参与，这会遏制商业交易与经济活动，欺骗型背信因为破坏外部对向性的信任关系，社会危害性较重，也不宜认定为背信犯罪，可能构成诈骗等罪名。此外，以非法占有为目的的故意型背信，可能构成监守自盗型的盗窃罪、侵占罪。因此，违背内部的信任关系，且不具有非法占有目的“故意型背信”，以背信罪惩治最为适当。[2]虽然彭华文教授不认为背信损害上市公司利益罪属于狭义上的背信犯罪[3]，但将该罪中的“背信”理解为“故意型背信”，无疑是非常贴切的。这反映出背信损害上市公司利益罪的主观方面是故意犯罪，不具有“掏空”上市公司目的的过失行为无法构成该罪；另外对上市公司财产具有非法占有目的的行为，也不属于该罪的打击范围，应以“职务侵占罪”等量刑更重的罪名进行惩治。其次，我们需要了解什么是“背信行为”。立法者在“背信损害上市公司利益罪”的罪状描述中，列举了六种背信行为方式，分别为：（一）无偿向其他单位或者个人提供资金、商品、服务或者其他资产的；（二）以明显不公平的条件，提供或者接受资金、商品、服务或者其他资产的；（三）向明显不具有清偿能力的单位或者个人提供资金、商品、服务或者其他资产的；（四）为明显不具有清偿能力的单位或者个人提供担保，或者无正当理由为其他单位或者个人提供担保的；（五）无正当理由放弃债权、承担债务的；（六）采用其他方式损害上市公司利益的。除了最后一项为兜底性质之外，三项涉及直接的利益输送，一项涉及违规担保、一项涉及不当处理债权债务。 最后，至于何为致使上市公司利益遭受重大损失，即该罪的具体刑事追诉标准，150万元是衡量能否入罪的一个关键金额。除了造成股票、债券、存托凭证或者国务院依法认定的其他证券被终止上市交易之外[4]，其他的利益输送、违规担保、不当处理债权债务的行为，造成直接经济损失在150万元以上的，即应予立案追诉。关于致使上市公司利益遭受特别重大损失，即应处三年以上七年以下有期徒刑并处罚金的具体情形，目前并无司法解释予以明确。从公开案例来看，判处被告人犯该罪获刑超过三年有期徒刑的，仅有陕西高院终审并维持原判的“王忠信背信损害上市公司利益案”[5]。王忠信为上市公司秦丰农业的董事长，法院认定其违背对公司的忠实义务，利用职务便利，将上市公司资金9,270,582.44元（系立案追诉标准的5倍以上）无偿转移至他人账户，且绝大部分金额无法追回，给上市公司的利益造成特别重大损失。法院最终判处被告人王忠信犯背信损害上市公司利益罪，判处有期徒刑四年，并处罚金一百万元。需要注意的是，本罪规定的兜底条款并不强制要求所造成的直接经济损失超过150万元。司法机关在适用兜底条款之前，一般会根据具体案情，判断嫌疑人在符合“违背对公司的忠实义务，利用职务便利”的前提之下，其的涉案行为是否与前述五项行为具有“同类解释相当性”。例如在新疆维吾尔自治区克拉玛依市克拉玛依区人民法院（2020）新0203刑初98号刑事判决书中，法院综合考虑涉案上市公司的大额资金被长期占用且因此受到证监会的调查和行政处罚，卷入诸多民商事诉讼并为此承担了账户被查封冻结、大额诉讼费用等危害后果，在并无证据证明被告人造成的直接经济损失超过150万元的情况下，认定被告人秦某操纵上市公司从事虚假借贷高额资金，或从上市公司借支高额备用金用于其实际控制公司及其关联公司，符合“采用其他方式损害上市公司利益”的规定，构成构成背信损害上市公司利益罪。三、背信损害上市公司利益罪与挪用类罪名的竞合关系背信损害上市公司利益罪的客观方面涵盖了挪用上市公司资金的情况，该行为也侵犯了上市公司对资金的占有、使用、收益的权利，可以认为，背信损害上市公司利益罪与挪用类罪名属于特别法与一般法的法条竞合关系，理由如下：一是因为该罪的主体身份更为特殊，限定为上市公司的董事、监事、高级管理人员、控股股东、实际控制人等。二是该罪所规定的第一款行为，即“无偿向其他单位或者个人提供资金、商品、服务或者其他资产的”，同样符合挪用公款罪或挪用资金罪的构成要件。三是该罪对危害结果有着更为严苛的规定，要求必须给上市公司利益造成重大损失；而挪用类罪名要求资金归个人使用进行非法活动的，或者挪用资金进行营利活动的，或者挪用超过三个月未还，对危害结果的规定，显然比背信损害上市公司利益罪更加宽泛。四是该罪与挪用类罪名所保护的法益均为公司的财产权利，但背信损害上市公司利益罪更侧重于对上市公司的利益保护，法益更为限缩，应当优先适用。鉴于挪用资金罪的法定刑最高为有期徒刑十五年，挪用公款罪的法定刑最高为无期徒刑，均重于背信损害上市公司利益罪的最高量刑有期徒刑七年。如果司法机关以挪用类罪名进行指控，而嫌疑人的行为完全符合背信损害上市公司利益罪的构成要件，辩护人应提出以特殊罪名论处的辩护意见，以实现轻罪辩护。例如，在上海市浦东新区人民法院(2007)浦刑初字第1521号案件中，主审法官在听取控辩双方意见后认为，被告人张某的行为同时符合挪用资金罪和背信损害上市公司利益罪的犯罪构成，从本案的犯罪性质来看，张某作为上市公司的董事长，无论是从主体身份还是犯罪的客观方面等均更为符合背信损害上市公司利益罪，对张某按照此罪定罪处罚更准确[6]。最终，法院判决被告人张某犯背信损害上市公司利益罪，判处有期徒刑二年，罚金人民币二千元。四、背信损害上市公司利益罪的辩护策略（一）关联交易性质之辩关联交易本身并不为《公司法》完全禁止，在授权或职权范围内开展的关联交易，如不违反“等价交换”等市场原则，或是因为决策失误造成公司损失的，因不具有通过不正当关联交易“掏空”上市公司的故意，不构成该罪。为了搞清关联交易的性质，辩护人应准确、全面梳理所谓“关联交易”的银行流水及最终流向。例如，如果相关资金流水是用于“走账”，且该部分资金最终回流至上市公司实际控制的银行账户，则表明所谓“关联交易”本身即不存在。“皮之不存，毛将焉附”，阐明关联交易的合法性之后，司法机关的相应指控，自然也就失去了成立的基础。（二）损害结果之辩本罪属于结果犯，必须“致使上市公司利益遭受重大损失”，达到《最高人民检察院、公安部关于公安机关管辖的刑事案件立案追诉标准的规定（二）》第十八条规定的情形，才予以立案追诉。辩护人如能否定案件中关于上市公司利益遭受损害的证据，可以提出不存在损害结果的无罪辩护意见。例如，在江苏省扬州市邗江区人民法院（2012）扬邗刑初字第0005号刑事判决书中，公诉机关指控被告人于某青作为上市公司的法定代表人、董事长，以上市公司名义向明显不具有清偿能力的控股股东等关联方提供担保的行为，违背对上市公司的忠实义务，致使上市公司利益遭受重大损失，应当以背信损害上市公司利益罪追究刑事责任。对此，辩护人提出了违规担保的风险已经化解，未给上市公司造成实际损失，不构成背信损害上市公司利益罪的辩护意见。法院经审理查明，控股股东以及于某青通过以股抵债或用减持股票款向债权人偿还的方式，清偿了全部债务，已经解除了上市公司的保证责任，因此被告人的行为并未给公司造成直接经济损失，不构成背信损害上市公司利益罪。（三）因果关系之辩即便存在损害结果，辩护人仍可以阐明被告人的行为与上市公司的损失之间不存在因果关系，实现无罪辩护的目的。例如，在北京市第三中级人民法院(2020)京03刑初170号刑事判决书中，被告人高某的辩护人提出相关回流资金并非购房款而是走账资金，不存在因不及时付款导致违约，进而造成公司损失的事实，故高某不构成背信损害上市公司利益罪的辩护意见。法院经过审理后认为，被告人高某确有真实购买涉案房产的意愿，并无通过购买涉案房产掏空上市公司的主观故意，并未以明显不公平的条件或者明显不合理的价格购买涉案房产，现有证据不足以证明被告人高某故意制造违约，导致殷某取得3500万元定金，且损失定金的事实发生在高某离职之后，与高某通过董事会购买涉案房产缺乏刑法上的因果关系。因此，公诉机关指控被告人高某构成背信损害上市公司利益罪的证据不足。法院最终采纳了辩护人提出的被告人不构成背信损害上市公司利益罪的辩护意见。小结由于背信损害上市公司利益罪对犯罪主体存在特殊要求，且仅限于对不具有非法占有目的的“故意型背信”行为进行刑事规制，故截止于本文完稿之日，能够查询到的法院公开判例未超过10件。然而，可以预料到的是，在证券市场“强监管”“零容忍”的背景之下，对资本市场违法犯罪的打击力度已经以肉眼可见的速度增强，该罪名的“出场”次数也明显增多，仅2021年就有4例上市公司高管涉嫌背信损害上市公司利益罪的案例被公开[7]。2022年9月9日，最高法、最高检、公安部、中国证监会联合发布的依法从严打击证券犯罪的五起典型案例中，即包含了“鲜某背信损害上市公司利益、操纵证券市场案”。最后，对于承担忠实勤勉义务的上市公司的董事、监事、高级管理人员，笔者建议，为防止涉嫌背信引起刑事风险，在进行商业决策时应始终具有合规意识，将个人决策通过提交股东大会或者董事会等内部治理程序，经过充分讨论和论证，转化为公司的集体意志，避免因个人独断而陷入被刑事追责的窘境。注释：[1]白话文翻译：人如果不讲信用，不知道他还可以做什么，就像大车没有车辕和横木衔接的活销，小车没有车辕与轭相连接的木销子，它将如何行走呢？[2]参见彭文华：《背信及其刑法规制》，载《当代法学》2020年第6期，第88页。[3]彭华文教授认为，违背非内部对向性信任关系的其他犯罪不属于背信犯罪。背信损害上市公司利益罪系违背劳务关系的犯罪，劳务关系并非以信用为纽带，而是一种劳动力与生产资料的结合关系，包含组织关系与人身关系，无法通过信用来调整。因此，背信损害上市公司利益罪不应归入背信犯罪。[4]鉴于2019年修订的《证券法》取消了暂停上市和恢复上市的制度，2022年版的《最高人民检察院
