# 案件移送管辖后被保全人权利救济困境1

来源：律角兽法律评论作者：廖一心 北京天驰君泰律师事务所上海分所律师，长期专注于民商事争议解决，熟悉破产清算业务，跟随团队期间参与处理多起破产清算、强制清算案件，熟悉财税法律事务。提起诉讼后，为保证胜诉判决顺利执行，申请诉中保全或者诉前保全是常见做法。但是，不排除存在为达不法目的，有人会选择在没有胜诉可能性的情况下，恶意提起诉讼并申请查封冻结被告财产，扰乱被告正常的生产生活秩序。正常情况下，被保全的当事人，可以通过对保全裁定申请复议或者提出异议进行救济，亦或者通过提供其他财产作为担保，要求法院解除查封关键财产。但是在案件被移送管辖的情况下，从案件移出到被新法院收入并进行审查、分配法官，这段时间移出法院经常会以无管辖权拒绝受理被保全人的异议申请，受移送法院又还没收到案件材料，也无从受理。此时，被保全人可能面临投诉无门的窘境。问题的产生在于移送管辖后，保全法院与有管辖权法院分离。无管辖权的法院是否能够继续对案涉财产采取保全措施，能否受理异议申请，在实践中存在一定争议。问题一：移送管辖后，原保全裁定如何处理？对于诉前保全案件移送后如何处理，法律规定比较明确。但是，对于诉中保全在移送管辖后，应该如何处理，法律并未对此作出明确规定。实务中有两种观点。第一种观点认为，无管辖权的法院无权采取保全措施，在移送管辖的同时应当解除查封。第二种观点认为，应当参照对诉前保全的规定，将是否解封的决定权交给受移送法院。观点一的缺陷主要在于原告需要承担在案件衔接中，被告转移财产的风险。据笔者了解，上海法院一般倾向于观点二，移送期间不解除保全措施，是否解封由受移送法院决定。观点二的缺陷在于，被告事实上承受了因移送管辖造成的保全时间的延长，恶意诉讼中被告的权益难以保护。另外根据《民事诉讼法司法解释》第一百六十五条：“人民法院裁定采取保全措施后，除作出保全裁定的人民法院自行解除或者其上级人民法院决定解除外，在保全期限内，任何单位不得解除保全措施。”受移送法院缺乏解除保全措施的法律依据。笔者认可观点二，在移送管辖的过程中，让保全财产处于“脱保”状态极易导致财产被转移，就算是解除查封后马上在受移送法院办理新的保全手续，也可能改变查封顺序，较大程度上影响了原告的权益。但是，在移送过程中也应当注意原、被告之间的利益平衡。至少在移送管辖的过程中，作出保全的法院不应当拒绝受理被保全人提交的保全异议和对执行裁定提交
