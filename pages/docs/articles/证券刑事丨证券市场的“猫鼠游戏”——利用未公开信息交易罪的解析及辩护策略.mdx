# 证券刑事丨证券市场的“猫鼠游戏”——利用未公开信息交易罪的解析及辩护策略

北京天驰君泰律师事务所上海分所合伙人按语：2019年以来，随着《证券法》的重新修订，以及《最高人民法院、最高人民检察院、公安部、中国证监会关于办理证券期货违法犯罪案件工作若干问题的意见》等司法解释的重磅出台，过去仅止步于行政处罚的证券类违法案件中，有相当数量被移送至公安机关刑事立案处理。相关数据显示，2021年人民法院依法严惩金融证券犯罪，审结操纵市场、内幕交易、非法集资、洗钱等犯罪案件1.3万件。这足见我国对证券类犯罪的打击力度之大，与十几年前相比，已不可同日而语。证券类犯罪是刑事领域最为复杂的分支之一。在目前证券市场“强监管”“零容忍”的背景下，辩护律师的充分参与，有助于证券类案件得到更客观、更公正、更全面、更专业的处理，从而依法维护委托人的合法权益。本文系专栏的第九篇：证券市场的“猫鼠游戏”——利用未公开信息交易罪的解析及辩护策略。硕鼠硕鼠，无食我黍！三岁贯女，莫我肯顾。[1]——诗经《国风·魏风·硕鼠》您应该对“老鼠仓”一词并不陌生，它翻译自英语“rat trading”，特指受托资产的管理人员在用资金拉升股价之前，利用个人实控账户在标的证券的低位价格建仓，待用受托资金将标的证券拉升到高位后，率先抛售个人仓位获利的损公肥私行为。该行为情节严重的，在刑法中会被评价为利用未公开信息交易罪。利用未公开信息交易罪与内幕交易、泄露内幕信息罪都规定在《刑法》第一百八十条，这表明内幕信息与未公开信息之间存在一定同质性，亦说明两罪的违法与责任程度相当。事实上，在《最高人民法院、最高人民检察院关于办理利用未公开信息交易刑事案件适用法律若干问题的解释》（以下称《利用未公开信息交易解释》）出台之前，法院在认定本罪“情节特别严重”时，甚至会直接参考2012年发布的《关于办理内幕交易、泄露内幕信息刑事案件具体应用法律若干问题的解释》。之所以发生上述情况，是因为法律对利用未公开信息交易的规制，总是迟滞于内幕交易。早在1997年《刑法》修订之时，法律条文即加入了“内幕交易、泄露内幕信息罪”。1998年颁布的《证券法》详细规定了内幕交易的认定标准、罚则等。然而，在2009年之前，证监会仅能以违反了《证券法》第四十三条有关禁止有关人员参与股票交易的规定，对利用未公开信息交易的人员进行行政处罚[2]。司法机关更是因刑法缺乏明确规定，而难以追究相关人员的刑事责任，否则将违反罪行法定原则。直至《刑法修正案（七）》在《刑法》第一百八十条“内幕交易、泄露内幕信息罪”之后增加第四款，规定了“利用未公开信息交易罪”之后，法律界才妥善解决利用内幕信息以外的未公开交易信息的交易行为，无法通过刑事罪名予以惩戒的问题。对此，有观点认为，在《刑法修正案（七）》中增加了该罪名，实质是以法律形式扩大了内幕交易罪中内幕信息的范围[3]。一、何为未公开信息即便是在2009年《刑法修正案（七）》出台之初，我国的法律体系仍未对“未公开信息”进行清晰的定义。直至2019年，《利用未公开信息交易解释》通过第一条明确，刑法第一百八十条第四款规定的“内幕信息以外的其他未公开的信息”包括：（一）证券、期货的投资决策、交易执行信息；（二）证券持仓数量及变化、资金数量及变化、交易动向信息；（三）其他可能影响证券、期货交易活动的信息。另外，鉴于证券市场在发展过程中，新的概念术语和交易模式不断涌现，为缓和立法的迟滞性，《解释》第二条特别赋予办案机关在难以认定时，可在行政主（监）管部门的认定意见的基础上，根据案件事实和法律规定认定“未公开信息”的权利。关于“未公开信息”的性质，学者之间众说纷纭。但都基本认同未公开信息具有“未公开性”和“价格敏感性”：“未公开性”具体指相关信息在交易实施之前并未公开，处于秘密状态。与同样具有未公开性的内幕信息有所不同，根据《证券法》第八十、第八十一条的规定，内幕信息尚未被投资者得知时，公司应当立即将相关情况向国务院证券监督管理机构和证券交易场所报送临时报告并予公告。而以交易信息为主的未公开信息，并不要求及时公布。“价格敏感性”要求未公开信息并非无中生有的虚假信息，其内容确实会对证券的交易价格产生波动，因此行为人在该信息公开之前获取，可以利用其获取高额利益或者减少损失。需要注意的是，司法机关在实务中，对价格敏感性的认定相当宽松，许多案件中的当事人和辩护人以未公开信息涉及的交易量较小、不足以对证券价格，特别是“蓝筹股”的股价产生影响为由开展辩护。然而，此类辩护意见基本上未得到过法院的采纳。由此可以看出，司法机关更加看重利用未公开信息交易行为对证券市场的信息公平性及从业人员的忠实性的破坏，而选择不去深究未公开信息的内容，究竟会对价格能够产生几成影响。也有观点认为，未公开信息还应具有“职务性”。即未公开信息应当是金融机构的从业人员以及有关监管部门或者行业协会的工作人员因职务便利获取的信息。此处的职务便利不仅包括利用职务身份直接获取未公开信息的具体内容，也包括业内人员利用工作环境的便利条件，以偷窥、窃取、交换等方式获取了未公开信息。利用未公开信息交易是从业人员对其忠实义务的违背，这也使得该行为具备了刑事上的可罚性，同时也决定了本罪的犯罪主体必须为特殊主体，一般主体仅能在共同犯罪的情形下构成本罪。二、如何认定利用未公开信息交易利用未公开信息交易本身是否成立，决定了罪与非罪。本罪与内幕交易、泄露内幕信息罪最为显著的不同在于，刑法并未将泄露未公开信息的行为单独列为一条独立的罪名以追究刑事责任，单纯泄露未公开信息的行为不构成本罪。换言之，当事人即便存在透露未公开信息并明示、暗示他人从事相关交易活动的行为，只要并未确凿证据证明被明示、暗示的对象在交易中利用了未公开信息，则当事人应当按照无罪处理。根据《刑法》，本罪的实行行为存在两种模式，一是“从事与该信息相关的证券、期货交易活动”，二是“明示、暗示他人从事相关交易活动”。简而言之，可分类为“交易行为”和“建议行为”。在后一种模式中，控方需要举证证明当事人的明示、暗示行为与他人的交易行为之间构成因果关系。根据《利用未公开信息交易解释》第四条[4]规定，认定“明示、暗示他人从事相关交易活动”时，应综合考虑行为人的职务便利条件、交易趋同程度、利益关联程度、交易习惯背离等。也就是说，需要证明未公开信息与信息产生后的交易活动存在较强的关联性。该规定的内容是对本罪刑事追诉经验的总结，意图拨开金融市场复杂的交易机制，直指利用未公开信息交易的行为特征。三、趋同交易与趋同率（一）趋同率是定罪的重要参考但非唯一指标利用未公开信息的交易又被称为“趋同交易”。趋同交易的比率（又称为“趋同率”），既可以从股票数量占比进行统计，也可以从交易金额占比来计算。以上这两种比率对案件都具有参考价值。趋同交易的比率越高，司法机关越倾向认定相关证券交易基于未公开信息开展，最终的获利与利用未公开信息之间存在因果关系。需要注意的是，趋同率的高低与能否定罪并无直接关系，司法机关在判断是否构罪时，需要全面审查当事人交易时是否利用了未公开信息的证据，趋同率只是其中一个指标，并不存在趋同率达到或未达到某一个数值，就一定构成或不构成犯罪，仍需结合其他证据综合判断。例如，在辽宁省朝阳市中级人民法院(2018)辽13刑初36号刑事判决书，被告人刘某某各个实控证券账户的的趋同股票比率均低于30%，最低的仅有13.89%，但相关成交金额均被计入，用于定罪量刑。（二）趋同率的“前五后二”计算方式司法实践中对趋同率的计算，一般采取“前五后二”标准，即将机构对某一只股票交易日期（T日）的前五个交易日（T-5日）至后两个交易日（T+2日）期间，当事人实控账户对相同股票的交易认定为趋同交易。该原则在审判领域的确立，来源于最高人民法院于2016年发布的指导案例“马乐利用未公开信息交易案”。在该案中，法院将马乐利用其实控账户，先于（1-5个交易日）、同期或稍晚于（1-2个交易日）其管理的“博×精选”基金账户的股票交易认定为“趋同交易”。法院之所以如此认定，是直接引用了深圳证监局对马某涉嫌利用未公开信息交易股票案件的调查结论，即证券监督管理部门在行政调查中即按照“前五后二”标准，认定马乐趋同交易涉及的股票数量，成交金额和获利金额的。而证券监督管理部门采用“前五后二”标准，是考虑到基金经理产生交易决策，至标的股票经基金公司内部审核进入股票池的时间，一般需耗时五日左右。基金公司的交易决策并非形成于机构交易当日，而是先于机构交易之日形成，因此需要采取“前五”的标准。至于“后二”，可以理解为交易发生后相关交易信息在二个交易日内足以被市场消化和接受。四、利用未公开信息交易罪的辩护策略（一）推定逻辑是否成立之辩根据上文，在利用未公开信息交易的认定上，仍然是以“职务”与“存在接触或特定关系”，来推定“当事人获取并透露了未公开信息”；以“交易行为趋同”来推定发生了“利用未公开信息交易”；以“交易风格、风险偏好、持仓资金发生重大变化”和“存在利益关联”来补强上述推定得出的结论。例如，在上海市高级人民法院(2013)沪高刑终字第5号刑事裁定书中，法院在综合被告人因职务事先掌握未公开信息、涉案证券账户满仓购买未公开信息涉及的标的股票、被告人系涉案证券账户的出资人及实控人之一、无证据认定证券交易系证券账户操作者李某君的擅自决定、此前被告人稳定供述向李某君发出了购买之类等事实的基础上，认定涉案证券账户中工商银行、建设银行股票系李某某指令李某君购买，即存在“明示、暗示他人从事相关交易活动”的涉案行为，最终判处被告人构成犯罪。既然司法实务中，控方对“明示、暗示他人从事相关交易活动”的认定仍然基于“推定”而非“证明”，相关指控必然存在逻辑上的“缝隙”。辩护律师在审查相关证据是否形成证据链并达到刑事案件证据标准之余，还可通过归谬或提出合理解释扩大逻辑“缝隙”，推翻控方依推定得出有罪结论。（二）未利用职务便利之辩本罪的构成要求未公开信息的获取行为，必须与行为人的职务存在关联。若未公开信息的来源与相关从业人员的职务行为无关，例如某退休人员因个人交情等原因进入金融机构后，通过偷听、套话、私自查阅等方式获知的未公开信息，不属于利用职务便利获取。另外，行为人通过分析公开信息并作出独立的交易判断，相关交易行为不构成本罪所要求的“利用职务便利获得的未公开信息”。事实上，许多基金经理的持仓情况和交易风格都是公开信息，普通投资者经过细致研究后仍然有可能与基金经理的交易形成趋同。如果行为人能对所实施相关交易活动作出合理解释，或者证明其行为符合过往交易习惯、专业判断的，并提供一定的客观证据对解释进行支撑，则行为人不应对自主决策的证券交易部分承担责任，从而取得一定的辩护效果。例如，在荆州市中级人民法院（2017）鄂10刑初68号刑事判决书中，被告人及辩护人提出对潮宏基、巨轮股份、长城集团和科士达等4只股票是被告林某某基于自己多年炒股的经验及其获取的公开的股市信息，对拟投资交易的股票进行选择所做出的分析判断，对上述股票的交易应排除出利用未公开信息交易的范畴。经过质证和辩论环节，公诉机关也同意将上述四只股票从指控的趋同交易股票范围内剔除。最终，法院在审理后因此调低了被告人的获利金额。（三）趋同交易计算标准之辩辩护人在审核趋同交易相关证据时，应避免僵化地适用“前五后二”标准。对于利用证券账户查询权限而获悉未公开信息的监管人员[5]，或者交易当天获悉指令的交易员，鉴于利用未公开信息交易全部发生在机构交易日之后，对“T-5日”至“T日”发生的交易应主张全部排除出趋同交易之列。在趋同交易金额的计算上，目前法院普遍将趋同交易期间内发生的单向趋同交易（买入金额）和双向趋同交易金额（买入+卖出金额）均应计入证券交易成交金额。有观点认为，计算双向趋同交易成交额违反禁止重复评价原则，加重了对被告人的刑罚的做法[6]。鉴于法律并未对趋同交易金额的计算方式有明确规定，从存疑有利于当事人的角度出发，可以主张将买入和卖出成交额分别进行统计，取两者中金额较大的数字认定为趋同交易金额。 小结2013年，我国利用未公开信息交易的监测分析系统正式上线，证券交易所可从海量交易数据中筛选可疑的“老鼠仓”交易行为，并报证监会查处，从而大大提高了对利用非公开信息交易行为的查处效率[7]。这也是用未公开信息交易刑事案件在2013年后集中涌现的直接原因。执法者与“老鼠仓”之间长年累月的“猫鼠游戏”，在监管部门全面应用大数据技术之后，已经成为一场实力悬殊的较量。得益于大数据的全面性和完整性，司法机关适用推定规则，对趋同交易进行认定的难度也大为降低，且趋同率较低也难以阻却犯罪的成立，因此相关从业人员一定要在利益诱惑面前筑牢底线，清醒认识到“老鼠仓”行为发生后，被查处已成大概率事件，莫要一时糊涂铸就千古恨。而辩护律师在代理利用未公开信息交易案件时，面对源自“大数据”系统的趋同交易明细以及监管部门的“认定函”等所谓“铁证”时，应挖掘冰冷数据与现实案情之间的“缝隙”，用合理解释进行“填充”，让当事人罚当其罪，甚至免受牢狱之灾。注释：[1]白话译文为“大田鼠，大田鼠，不许吃我种的黄米！多年辛勤伺候你，你却对我不照顾。”[2]2019年修订的《证券法》也终于在新设的第五十四条，对利用未公开信息交易的行为明确予以禁止性规定：禁止证券交易场所、证券公司、证券登记结算机构、证券服务机构和其他金融机构的从业人员、有关监管部门或者行业协会的工作人员，利用因职务便利获取的内幕信息以外的其他未公开的信息，违反规定，从事与该信息相关的证券交易活动，或者明示、暗示他人从事相关交易活动。利用未公开信息进行交易给投资者造成损失的，应当依法承担赔偿责任。[3]王晓磊：《利用未公开信息交易罪中未公开信息的认定》，载《中国检察官》2019年第29期，第6页。[4]第四条
