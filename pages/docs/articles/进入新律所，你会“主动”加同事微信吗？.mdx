# 进入新律所，你会“主动”加同事微信吗？

“在22年2月14日之前的这段时间，我的精神内耗越积越多，困惑，焦虑，最终抱着试一试的态度走进曹大夫的诊所。”军超来找我，其实还有一个原因，我在2021年年底的时候建了一个读书群，邀请过他参与，有大概半年的时间，每天早晨五点钟我会准时在里边发读书感悟。现在因为忙，坚持的也不好。律师是与人打交道的职业，与他人之间的连接只是开始，链接才是目标。一定要有意识的去链接人。人与人之间信任的建立固然需要一个过程，但是不是一定要用这么久的时间？做律师一定要面对面接触人，包括同事，很多想法都是面对面闲聊的时候激发出来的。也算是无用之用有用的一个注脚吧。别怕麻烦，别怕拒绝，多找同事闲聊，多找合伙人请教，连接→链接，需要付出的，如同认识→结婚，一定会有一段时间的磨合。有的人，走近了，你就会发现原来那么投缘。有的人，最初挺谈的来，日久天长，又渐行渐远。罗翔老师说：“误解是人生的常态，理解是稀缺的意外。”当我走过了一些律所，见了一些同行后发现，共识不只来自一见钟情，更来自于持续的影响和培养。独木不成林，想成为优秀的律师，就必须培养人，团结人，“凑人”组团、合伙是不长久的。相比其它行业，律师行业创业成本比较低，流动性自然比较大，正常的流动，当然是好事。只怕，为了转所而转所，为了开所而开所。律师圈里有人笑谈，这个年代，合伙比结婚都难。诚然，不同的律所各有千秋，每个人都会有自己的审美，不同的阶段也会有不同的选择。一家律所能够同时具备有一定规模、有一个与时俱进的掌舵人、有一些持续活跃的优秀合伙人、有一套以共生为价值导向的制度、有一个优秀的管理团队、有一个高效勤勉的行政保障团队、有清晰的战略、符合人性的文化、比较便利的办公条件，并没有那么容易。面对年轻人离婚率高企的现象，有人评论，老一辈的人碰到问题总想着修，年轻人面对问题总想着换。实际上，没有理想的工作，只有不断追求的理想。“换”永远解决不了自身的问题，所以，换不来共识，只能换自己。03悲观或许是远见，乐观往往是智慧很多时候，打败我们的，不是困境和疾病，而是我们的心态。自己没本事的人，往往喜欢否定别人，我们应该学会与他人的优点相处。我们真的应该习惯性的多去看积极的一面，要透过平台看到背后一个一个具体的人。尤其在大所中，可能百分之九十的人都是值得交往的，只有少数人是极其不对脾气的。不能因为少数人的一些行为，渐渐走向自我封闭，放弃掉和原本值得交往的朋友的正常相处。走入职场，总会有一些人会最早看见你，帮助你；还有一些人，需要你做出点成绩，也会看上你，帮助你；还有一些人，一开始可能根本看不见你，随着你影响力的提升，慢慢的也会发现你；甚至，有一些有过不愉快的人，也会转身帮助你。这是一种正常的现象，只要你足够优秀，你就会发现朋友就会越来越多，因为与你合作彼此都最有利。交朋友的过程，是价值交换的过程，也是凝聚共识的过程，而这，也终将汇聚为成就你的事业的洪流。04人际交往的五句话需要知行合一《青年律师执业、进阶必修技能》一书中提到的人际交往五句话，我很赞同。第一，发自内心地尊重别人；第二，恰到好处地赞美别人；第三，顺时顺势地成就别人；第四，善意以待别人的善良；第五，注意把握交往的边界。发自内心的尊重别人这点很不容易，中国人讲究“慎独”，就是在一念起处，在只有自己所知道的地方为善去恶。我们应该时常问问自己是否真的足够尊重别人。机会不一定都到了专业能力最好的人手里，往往到了最主动的人手中。通常，同事与你合作往往因为两个理由，一个属于感性，觉得你人还不错；一个属于理性，觉得你有解决这个问题的学识和能力。但如果你是一个年轻律师，更可能因为第一个理由给你合作的机会。此处特别想提醒一下年轻律师朋友，“我”来解决问题，与“我们”一起来解决问题不一样，要跳出自己来解决问题。给你机会就要先大胆接下来，接下来以后你可以找人问，找多个人问，不要觉得不会做就急忙拒绝，你啥都会了还是年轻律师吗？有时候看中的就是你的态度，谨慎是必要的，方法也很重要。来到段和段之前我也仅仅认识一个人，慢慢的就有了很多的朋友，细想，与十位同事之间不也是从陌生人到熟悉的人，再到成为彼此信任的人吗？在职场中，同事是我们最亲的人，最近的人，是最近的市场。大部分青年律师的第一个案件很可能是同事介绍的。是最容易得到的案件来源处，如果我们连主动认识他们的勇气都没有，面对陌生人，又怎么能够赢得他们的信任与委托呢？此处推荐阅读：曹振华：青年律师如何结交 高人”曹振华：青年律师需要持续被看见不要总为失败找借口，但要多为进步找方法。吃饭要一口一口吃，伸手要一个一个伸，朋友要一个一个交，与其想着一炮而红，不如相信水滴石穿。你愿意善良以待每一个人，愿意发自内心赞美别人，愿意对别人的善意及时反馈，保持分寸感，愿意一个一个交朋友的心态去与人相处，一定会有越来越多的朋友。好习惯是最落地的成长法则。我们大部分人都高估了自己的坚持能力。在大部分的自我较量中，只有两个制胜法宝：第一个是去做，第二个是坚持去做。按着这五句话去坚持链接人，久久自然得益。05勇敢面对自己的不理解律师是专业人士，我们需要管理好情绪，保持情绪独立。职场中，有时候面对自己的不理解，也要深入思考背后的原因，多点宽容。我们时常听到一种抱怨，某某新来的，见了同事低着头走路。有一次我对一位同事说，中国缺乏系统的职前培训，实习律师通常都是一张白纸，尚未经历职业化的训练，有的人天性偏热情一些，与陌生人熟络的就会快一些。有的人天性内敛一些，跟你不熟悉，有点社恐其实是很正常的一件事儿。我们年轻的时候也未必比他们强多少，如果我们以一种永远先于别人伸出你温暖的双手的态度来处理的话，或许可以主动自我介绍，并邀请他到办公室喝茶，下次见你大概率就不会再低头了。有些看似很LOW的行为，或许仅仅是从来没有人给他提过应该怎么做。青年律师是行业的未来，如果我们都能对新人多点宽容和理解，多点雪中送炭，少点莫名指责，以“只影响不改变，唯愿以诚相待”的态度去处理人际关系的话，你的人缘会好很多。青年律师在执业之初的迷茫中也会多感受到一些温暖。一个感知到过爱的人，也更容易去爱别人，而爱越分越多。06宽容以待他人的不理解工作中，即使面对他人的某些不理解，作为专业人士，我们也要时刻警醒，要习惯于反人性，反射弧慢半拍，作出职业化的反应，职业化不止是法律层面的专业化，也包括其他不限于沟通技巧等等的专业化。尤其是在面对一些信任基础尚未完全建立、价值感尚未完整塑造的客户，人的第一反应一定是，你对我不理不睬，我也视你为空气。我就被这十个人中的两三个怼过，并不影响我们成为好朋友。律师要成为人精。何谓人精？了解人性，顺应人性。如果做事都凭喜怒，那只是一个普通人，怎么能称为“专业人士”。多一分宽容和理解，多给别人一两次机会，天塌不下来，喜欢你，信任你需要一个过程。对同事如此，对客户也是同理。一枝独秀不是春。独立律师与律师独立是两码事，独立律师不是要独善其身，恰恰是要在独立基础上加强与其他律师的合作。律师是挺孤独的一份职业，我们要努力寻找一些同行者。业务分工越来越细了，单打独斗几乎很难胜任较为疑难复杂的案件。在寻找合作者的过程中，第一印象当然重要，毕竟凭第一印象看人是人性，但真正能够让大家走到一起的一定是理念差不多，步伐差不多，努力程度差不多，大家在彼此自由的前提下，积攒情谊，互相成就。   07向他人释放善意，就是给自己机会谁与谁投缘，谁与谁合作，谁与谁同行，说到底，就是个概率事件。一直觉得，当下的青年律师普遍成长困境下，最稀缺的素质是勇敢。不要顾虑太多，加个微信仅仅是一个开始，只是增加了某种可能，但没有开始就不会有继续。主动加个微信是件小事儿，然而生活是需要去发现的，发现不仅仅是看到，因为很多时候我们会视而不见，会熟视无睹，发现是深刻地看到，而且知道由此可能会带来什么。人际交往中，虽然大部分的人是被动的，但是每个人内心都是渴望被重视、被交流的，只要我们主动，很多机会都会悄然而来。已故的张思之律师说：“真正的律师，应是一团火，从点燃到熄灭，持续发着光，散着热。”律师是知识分子，大多数人都是明是非、知好歹的，时间从来不语，但可以见证人心。在大家的感慨声中，夜渐渐深了，岁月温柔的撞了我们一下，又悄然隐入尘烟，当然并未尘埃落定，明天一定会更好。无讼小编：如果您觉得这篇文章还不错，欢迎转发分享、点赞收藏，您也可以在下方评论区留下自己的观点，和大家一起讨论。主编：靖力责编：林慧审核：刘逸凡
