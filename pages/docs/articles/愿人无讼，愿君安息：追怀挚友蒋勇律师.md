# 愿人无讼，愿君安息：追怀挚友蒋勇律师1

 
来源：IDG资本
 
 
距离第一次见到蒋律师，已经过去7年了。第一面，他就给我们留下了非常深刻的印象，不仅为人热情、慷慨激昂，且充满了对创业和理想的执着。蒋律师很快就感染了在场的所有人，大家一致认定这会是我们志同道合的好朋友。
 
后来，蒋律师常邀大家去他的院子里畅谈。去年这个时候，蒋律师院子里的杏子熟了，他摘下来赠与我们。如今，果实依旧将熟，而这位热忱的朋友，却已然离我们而去。
 
 
过去，蒋律师经常和大家一起讨论行业发展与可能性。在创业期间，他不甘心固有模式的冗沉，于是不断探索、创新，尝试推动律师业的数字化进程。
 
这不是一件容易的事，他却坚持迎难而上。他像一座灯塔，努力地照亮着行业，照亮着身边人。
 
毫无疑问，蒋律师是一个理想主义者。正如那句话：我们坚持一件事情，并不只是因为这样做了会有结果，而是坚信，这么做是对的。蒋律师义无反顾为了崇高的目标出发，这种品质是当下时代稀缺的，因稀缺而显得弥足珍贵。
 
在2017年中国经济潮流人物的颁奖典礼上，他曾动情地说过：“我深知，这个奖项属于无讼和我一起创业的小伙伴们，属于全国超过30万律师，属于全体法律人。”尽管荣誉加身，他却非常自谦，所谓君子之风，山高水长，蒋律师亦当之无愧。
 
“但愿人无讼，何妨我独闲”。举世浮沉，大浪滔滔，蒋律师从不随波逐流，他选择的是公义，是为社会尽责，是献身于真理。他投入到伟大事业中的一生，令人钦佩。
 
往事依依，千言万语，此间难尽。现在，一眼望去，院子里林木已深，而先生留下的愿景，也将枝繁叶茂，不断焕发生机。
 
此刻，尽有亲人掩面与朋友伤怀，且让我们忍住哽咽，痛悼这位曾经携手相伴的挚友。而他所秉持的精神，将鼓舞我们更有力量地前行！
